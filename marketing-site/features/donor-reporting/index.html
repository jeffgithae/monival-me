How it works

Set it up once. Let it run every cycle.

  • Choose a reporting period — the report pulls every approved result and activity inside that window.
  • Pick a cadence — daily, weekly, monthly, or quarterly, with a specific day of month for recurring cycles.
  • Add recipients — the report is generated and emailed automatically when it's due, with no one needing to remember.
  • Trigger on demand — need it early for a surprise donor call? Generate it manually any time.
